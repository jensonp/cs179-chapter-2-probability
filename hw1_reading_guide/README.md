# HW1 Reading Guide

Use this guide with your local `cs179-hw1.pdf`.

Do not read the whole chapter first. For each homework problem:

1. Open the listed note.
2. Start at the exact heading named below.
3. Read until the stated stop point.
4. Stop reading and solve that homework problem immediately.
